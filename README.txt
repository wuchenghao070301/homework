3D 汉诺塔益智游戏
=====================
一、游玩方法
1. 使用滑块/输入框设置 3-8 个圆盘
2. 点击顶部圆盘选中，再点击目标柱子移动
3. 遵守规则：小圆盘只能放大圆盘上
4. 全部移到最右侧柱子即通关
5. 支持重置、自动求解、步数统计

二、使用的数组方法（必选）
1. push() / pop()：圆盘栈移动
2. forEach()：渲染柱子与圆盘
3. slice()：数组重置（初始化）
4. at(-1)：获取栈顶元素
5. length：判断栈高度/胜利条件

三、核心功能
✅ 3D视觉效果（透视、渐变、阴影）
✅ 点击选择+点击移动（推荐方案）
✅ 非法移动警告+抖动动画
✅ 通关庆祝动画
✅ 自动递归求解
✅ 响应式适配
✅ 步数统计
✅ 游戏重置

四、已知问题
无功能性BUG，自动求解延迟固定800ms
